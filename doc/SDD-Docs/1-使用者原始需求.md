會員註冊與登入系統

1. 註冊功能
* 輸入欄位：姓名、Email、密碼、確認密碼
* 密碼規則：8-20碼，必須包含英文大小寫與數字
* 註冊後發送6位數驗證碼到Email（有效期5分鐘）
* 必須通過Email驗證才能完成註冊

2. Email驗證功能
* 手動輸入6位數驗證碼
* 驗證碼5分鐘後自動過期
* 提供「重新發送驗證碼」功能（60秒冷卻時間）
* 驗證成功後才能建立帳號

3. 登入功能
* 使用Email和密碼登入
* 登入成功後自動儲存會話
* 支援自動登入（下次開啟APP直接進入）

4. 登出功能
* 清除會話資料
* 返回登入頁面

---

記帳主頁功能

1. 財務概覽
* 顯示總收入金額
* 顯示總支出金額
* 顯示結餘（收入-支出）

2. 新增收支記錄
* 選擇類型：收入 或 支出
* 選擇類別：
* 支出：餐飲、交通、購物、娛樂、醫療、教育、其他
* 收入：薪資、獎金、投資、兼職、其他
* 輸入金額（必填）
* 輸入備註說明（選填）
* 自動記錄時間

3. 交易記錄列表
* 顯示所有收支記錄
* 顯示內容：類別、金額、備註、時間
* 最新記錄顯示在最上方
* 收入顯示綠色+金額，支出顯示紅色-金額

4. 用戶資訊
* 顯示用戶姓名
* 個人化問候語


---

系統限制

註冊與登入
* 不支援社群帳號登入（Google、Facebook等）
* 不支援忘記密碼功能
* Email驗證不支援點擊連結驗證（僅手動輸入驗證碼）
* 不顯示密碼強度提示
